How to Install?

# Upload the contents of the upload folder to your root directory. This will not overwrite any files.

# Go to Admin -> System -> Users -> User Groups , and give appropriate permission to extension/hb_sitemap

# Now Go to admin -> Extensions -> Click on XML Sitemap, This will be auto installed .

# Now you can edit parameters in settings tab and click on save button

# Click on appropriate Generate button to generate XML Sitemap
